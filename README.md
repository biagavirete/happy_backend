## Getting started

**Installing**
>Cloning the repository

```bash
$ git clone https://github.com/biagavirete/happy_backend.git
$ cd happy_backend
```

**Running**
> Installing dependencies

```bash
$ yarn
```

> Running API. 

Create the database and the tables and then run the project:

```bash
$ yarn typeorm migration:run
$ yarn dev
```
